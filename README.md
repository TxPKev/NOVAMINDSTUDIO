## Projects

### AriNet — Deterministic Offline AI System

A complete AI assistant architecture that runs 100% locally.

| Component | Technology | Function |
|-----------|-----------|----------|
| Speech Recognition | Whisper | Understands spoken language |
| Voice Synthesis | XTTS-v2 | Natural German voice output |
| Vision | Florence-2 | Sees and interprets the environment |
| Language Understanding | DistilBERT | Intent recognition & context |
| Brain | llama.cpp / GGUF | Reasoning, optional LLM |
| Orchestration | 160+ specialized agents | Deterministic pipeline |

**Key properties:**
- Every decision is traceable — no black box
- Identical input → identical output, always
- Zero cloud calls during operation
- Runs on consumer hardware (RTX 3080+)
- Full German language support

> *"Ich baue bis Ari neben mir steht."*

### Guardian — Offline Security Framework

Patent-pending security architecture for connected hardware devices.

**Core capabilities:**
- Port integrity monitoring — reaction time **≤100ms**
- Presence and state evaluation from local device signals
- Energy state control and intelligent power management
- Local encrypted event fingerprints for audit trails
- Fully offline — no network required at any point

**Target domains:**
- Consumer electronics & smartphones

- Enterprise hardware security

Designed for licensing to global manufacturers of connected hardware devices.
